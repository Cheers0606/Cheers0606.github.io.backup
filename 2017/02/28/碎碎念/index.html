碎碎念

今天正式离职了，以为心里会很慌，或者是有些激动，结果一晚上下来，心里却没有什么波澜。

从入职说起

2016年3月15号，我跟上海某公司签了实习合同，当时做的项目是某公司的卡贴后台，虽然是上海公司，办公地点却是在广州这边（老总在广州有个小创业公司，同时单人上海那边公司的技术总监一类的职位）。当时我期待的薪资是7.5K，那边给出的答复是以上海最高的薪资水准，转正后给我6K，实习3个月，实习期间薪资为2.5K。由于当时刚毕业，准确来说是还没毕业，但是看到周边的人都陆陆续续找到工作了，讲真心里有些着急，而且我又是比较向往外地的，像魔都这种地方，年底如果有年会什么的可以过去逛一圈，想想也是棒棒哒，所以也没有多想就答应了。
到了6月底转正的时间，上海那边却说，他们决定把技术这一块的开发维护工作也转移到上海去，所以我算是被炒了，老板不知道是觉得之前答应了我可以转正又没转成还是像他说的那样“觉得你工作能力还可以”，叫我留在广州他的公司上班，薪酬同样是6K。忘了当时是怎么想的了，总之最后是留了下来。

关于公司

在这家公司上班上班，跟想象中还是不太一样。我以为初创会很有激情，亦或是大家会像兄弟一样感情会很好。然而，事实是，每天要么忙得不可开交，根本没有心情聊天，要么是闲得发慌，不好意思稳别人在干嘛..总之，给我的感觉就是，没有集体感，没有团队氛围。在初创公司上班的好处，我想应该是上班时间的半自由吧。（早上9点半上班，一般10点前去到都可以，平时有什么事需要请一两天假，基本也是批的）

关于离职

离职的原因，其实并不是我跟老总说的“我要回家乡发展”之类的，其实一方面是觉得薪资不够满意，另一方面是像上面说的，没有集体感，没有激情，还有就是感觉公司前景不太光明。之所以跟老板说要回家乡发展所以辞职，不知道算不算是一种善意的谎言呢。（总不能直接说我觉得公司前景不好，工资太低这么直白吧┑(￣Д ￣)┍）

重新开始

我在想，离职之后，是要给自己一点时间浪一下呢，还是赶紧找下一份工呢。
前些天看了美剧【黑客军团】，里面多次提到美国梦这个概念。其实也就是对未来有信心，可以透支未来。但我现在的状况，透支得起吗。
桢莹说“我也想任性一把，可是我不敢”。什么是任性呢，在我看来，做得不开心没有乐趣坚持下去又有什么用呢，人生短短数十年，要在自己不喜欢的环境下呆那么久吗。至于敢不敢，下了决心自然就敢了嘛。
3月4日，易迅科技招java人员，我想我是回去碰碰运气的，说不定就成了呢。不成也没所谓啊，总会有下家的吧。
加油吧
